🔧 Como Limpar o Cache do Navegador - Transmill

⚠️ IMPORTANTE: O navegador está usando código JavaScript antigo (em cache). Você PRECISA limpar o cache para ver as correções!

📱 Método 1: Hard Reload (MAIS RÁPIDO)

Windows / Linux:
Pressione Ctrl + Shift + R ou Ctrl + F5
Mac:
Pressione Cmd + Shift + R

🗑️ Método 2: Limpar Cache Manualmente

Chrome / Edge / Brave:
  1. Pressione Ctrl + Shift + Delete
  2. Selecione "Todo o período"
  3. Marque APENAS: "Imagens e arquivos em cache"
  4. Clique em "Limpar dados"
Firefox:
  1. Pressione Ctrl + Shift + Delete
  2. Selecione "Tudo"
  3. Marque APENAS: "Cache"
  4. Clique em "Limpar Agora"

🔄 Método 3: DevTools (Para Desenvolvedores)

  1. Pressione F12 para abrir DevTools
  2. Clique com botão direito no ícone de Recarregar
  3. Selecione "Esvaziar cache e atualizar de forma forçada"

🧹 Método 4: Service Worker (Automático via JavaScript)

✅ Como Confirmar que Funcionou?

1. Após limpar o cache, faça login novamente
2. Abra o Console (F12)
3. Vá em: Pessoas → Tipo de Veículo → Tabela FIPE
4. Clique em "Importar da FIPE"
5. Verifique no Console a URL deve ser:
✅ POST /api/labelview/fipe/importar-completo
❌ NÃO deve ser: /api/api/labelview/...

🚨 Se AINDA aparecer /api/api/...

Faça TODOS estes passos:
  1. Feche TODAS as abas do Transmill
  2. Pressione Ctrl + Shift + Delete
  3. Limpe: Cache + Cookies + Dados de sites
  4. Feche o navegador COMPLETAMENTE
  5. Abra novamente e faça login

📞 Suporte

Se após seguir TODOS os passos ainda aparecer erro /api/api/, tire um print do console (F12) e envie para análise.
